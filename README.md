# 🤖 AI Image Classifier using Teachable Machine

## 🔗 Live Demo
https://garimagarg-teachable.netlify.app

## 📌 About the Project
This project is built using Google Teachable Machine.  
It is a machine learning web app that classifies images in real-time using a webcam.

## ✨ Features
- Real-time image classification  
- Custom trained ML model  
- Interactive UI  

## 🛠️ Tech Stack
- Teachable Machine  
- JavaScript  
- HTML  
- CSS  

## 📂 How to Run
1. Clone the repo  
2. Open index.html in browser
